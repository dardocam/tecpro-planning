# 🧪 Trabajo Práctico N1: Laboratorio (Simulado)

## 📝 Redes LAN – Diseño, Configuración y Análisis

## 🎯 Objetivo

Comprender cómo funciona una red LAN mediante el diseño, la simulación de conexiones y la asignación manual de direcciones IP.

---

## 📌 Modalidad

* Individual o en parejas
* Resolución en **cuaderno**
* Trabajo guiado paso a paso

---

## 🧠 Situación problemática (contexto real)

> Una escuela necesita armar una red local en un aula de informática.
> Debe conectar:

* 1 router
* 3 computadoras
* 1 impresora
* 2 celulares

Todos deben poder comunicarse entre sí y acceder a internet.

---

# ✏️ Parte 1: Diseño de la red

### 🔹 Consigna

Dibujar un esquema de la red LAN.

📌 Debe incluir:

* Router (centro de la red)
* Todos los dispositivos
* Tipo de conexión:

  * Cable (Ethernet)
  * Inalámbrica (Wi-Fi)

👉 Ejemplo de referencia (NO copiar, solo guiarse):

```
        [Celular]   [Celular]
            \         /
           ( Wi-Fi )
              |
[PC1]---\
[PC2]----( Router )----[Impresora]
[PC3]---/
```

---

# 🌐 Parte 2: Identificación de componentes

Completar en el cuaderno:

| Dispositivo | Tipo (Final/Red) | Conexión | Función |
| ----------- | ---------------- | -------- | ------- |

📌 Ejemplo:

* Router → Dispositivo de red → Wi-Fi/Cable → Conecta la red

---

# 🔢 Parte 3: Asignación de direcciones IP (simulación)

👉 Explicación breve para alumnos:
Una dirección IP identifica a cada dispositivo dentro de la red.

---

### 🔹 Consigna

Asignar direcciones IP a cada dispositivo siguiendo estas reglas:

* Red base: **192.168.1.0**
* Router: **192.168.1.1**
* Los demás dispositivos:

  * No repetir IP
  * Seguir el mismo rango

---

### 📋 Completar:

| Dispositivo | Dirección IP |
| ----------- | ------------ |
| Router      | 192.168.1.1  |
| PC1         |              |
| PC2         |              |
| PC3         |              |
| Impresora   |              |
| Celular 1   |              |
| Celular 2   |              |
