# 🧑‍🏫 CLASE 1 – ¿Qué es HTML5?

## 🎯 Objetivo de la clase

Que los estudiantes comprendan:

* Qué es HTML5
* Para qué sirve
* Cómo se estructura una página web básica

---

# 🟦 PARTE 1 – DISPARADOR (10 min)

👉 Pregunta inicial:

💬 **“¿Cómo creen que está hecha una página web por dentro?”**

Escuchar respuestas como:

* “código”
* “programación”
* “botones e imágenes”

👉 Conclusión guiada:

> Una web está construida con **lenguajes**, y el principal es HTML.

---

# 🟦 PARTE 2 – EXPLICACIÓN TEÓRICA (15 min)

## 🌐 ¿Qué es HTML5?

HTML significa:
👉 **HyperText Markup Language**

📌 Es un lenguaje que sirve para:

* Estructurar contenido
* Organizar información
* Definir partes de una web

---

```html
<!DOCTYPE html>
<html>
<head>
  <title>Mi primera página</title>
</head>
<body>
  <h1>Hola mundo</h1>
</body>
</html>
```

---

## 🧠 Explicación paso a paso:

* `<!DOCTYPE html>` → Indica que usamos HTML5
* `<html>` → Contenedor de toda la página
* `<head>` → Información (no visible)
* `<body>` → Contenido visible

---

## 🎯 Idea clave:

👉 HTML **NO diseña**, solo **estructura**

(El diseño se hace con CSS → próxima clase)

# 🟩 PARTE 3 – CONSTRUCCIÓN GUIADA (20 min)

👉 Actividad paso a paso (hacer en vivo):

### 1. Crear archivo:

```
index.html
```

### 2. Escribir estructura base

### 3. Agregar contenido:

```html
<body>
  <h1>Mi primera web</h1>
  <p>Esta es mi primera página en HTML</p>
</body>
```

---

## 💡 Explicar etiquetas básicas:

* `<h1>` → Título
* `<p>` → Párrafo
